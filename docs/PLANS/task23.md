# Задачи: Тестирование динамической загрузки ONNX-моделей

- [ ] Написать Python-скрипт (`scripts/train_test_models.py`) для генерации синтетического датасета и его нормализации.
- [ ] Обучить 2 модели с разным размером входа: Random Forest (15 фичей) и Logistic Regression (5 фичей).
- [ ] С помощью `skl2onnx` экспортировать модели в `.onnx`.
- [ ] Автоматически сгенерировать файлы `scaler_params.json` (с указанием названий классов и списка фичей) для каждой из моделей.
- [ ] Провести тестирование в GUI программы, переключаясь между моделями и проверяя корректность детектирования и передачи данных в интерфейс и БД.
