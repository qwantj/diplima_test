# Отчет о соответствии программного комплекса заданию на ВКР

**Тема:** Разработка программного комплекса обнаружения атак типа «отказ в обслуживании»  
**Студент:** Дерюга А.А. (А-08-22)  
**Дата проверки:** 10.05.2026

Ниже приведено сопоставление требований из задания на выпускную квалификационную работу (ВКР) с фактической реализацией в программном комплексе.

---

## 1. Компоненты программного комплекса

| Требование задания | Реализация в проекте | Файл/Модуль |
| :--- | :--- | :--- |
| **Программа сбора и анализа данных** | Реализована как консольное приложение `ddos_collector`. | `src/collector_main.cpp` |
| **Программа мониторинга и управления** | Реализована как GUI-приложение на Qt `ddos_monitor`. | `src/monitor_main.cpp`, `src/monitor_ui/` |
| **База данных** | Используется PostgreSQL 14+ для персистентного хранения. | `src/common/DatabaseManager.cpp` |

---

## 2. Функциональные возможности

| Требование задания | Реализация | Компонент |
| :--- | :--- | :--- |
| **1. Захват трафика в реальном времени** | Использование Npcap и библиотеки PcapPlusPlus. | `TrafficMonitor` |
| **2. Загрузка и анализ дампов (PCAP)** | Поддержка режима `--pcap` и команды `load_pcap`. | `TrafficMonitor` |
| **3. Извлечение ключевых признаков** | Агрегация пакетов в окна (2с) и расчет 8 признаков CIC-DDoS. | `FeatureExtractor` |
| **4. Классификация через ML** | Инференс ONNX моделей (Random Forest / MLP). | `ModelInferencer` |
| **5. Сохранение событий в БД** | Асинхронная пакетная запись в PostgreSQL. | `DatabaseManager` |
| **6. Визуализация в GUI** | Дашборды, графики PPS, журналы событий в Qt-интерфейсе. | `ddos_monitor` |
