# 📋 План развития проекта DDoS Detector

Подробный план и TODO на основе [RECOMMENDATIONS.md](file:///c:/Dev/CXX/diploma_test/RECOMMENDATIONS.md). Охватывает все направления: производительность, стабильность, UI/инфографику и функциональные расширения.

## 🏗 Архитектура проекта (текущая)

```
src/
├── collector_main.cpp        — точка входа коллектора
├── monitor_main.cpp          — точка входа монитора (Qt)
├── core/
│   ├── DetectionEngine       — оркестратор: захват → фичи → инференс → результат
├── ml/
│   ├── ModelInferencer        — ONNX Runtime обёртка
├── network/
│   ├── TrafficMonitor         — pcap захват пакетов
│   ├── FeatureExtractor       — извлечение 83 фичей из потока
│   ├── PcapDumper             — запись pcap-дампов
├── common/
│   ├── DataBridge / Protocol  — TCP-протокол коммуникации collector↔monitor
│   ├── DatabaseManager        — запись в PostgreSQL
│   ├── TcpServer / TcpClient  — сетевой транспорт
├── monitor_ui/
│   ├── DashboardWidget        — главный дашборд с графиками
│   ├── LogWidget              — виджет логов
│   ├── SessionWidget          — управление сессией
```
