# План доведения UI/UX до идеала (Grafana / Yandex Monium Style)

На основе анализа исторических документов из архива `docs/PLANS/`, спецификации `UI_Reconstruction_Spec.md`, идей интеграции стилистики *Yandex Monium / Grafana* и **детального визуального анализа скриншотов (monium*.png, grafana*.png, и старого UI)**, выявлены конкретные недостатки текущей реализации и составлен пошаговый план улучшений.

---

## 0. Анализ референсов (Визуальный аудит)

*   **Grafana (grafana*.png):** Интерфейс строится на базе четко очерченных "карточек" (Panels). Темный фон приложения контрастирует с чуть более светлым фоном карточек. У каждой карточки есть заголовок. Графики (Area и Line) имеют тонкую сетку, а кольцевые диаграммы (Donuts) — крупный текст по центру с легендой в виде небольших цветных меток сбоку или снизу.
*   **Yandex Monium (monium*.png):** Идеально чистый дизайн (clean data). Отличительная черта — **Сводные тултипы (Smart Tooltips)**. При наведении на график появляется белая плашка, где в столбик перечислены все метрики (с цветным квадратиком), их текущее значение, а в самом низу — сумма.
*   **Старый UI (media__*.png / DDoS Dashboard*.png):** Верхняя панель (Header) выглядит как грубый сплошной зеленый прямоугольник. Таблицы (например, `Event Log`) имеют скучный стандартный вид `QTableWidget` без чередования строк. Графики слипаются с краями окна. Кольцевые диаграммы слишком малы.

## 1. Редизайн Главного Дашборда (DashboardWidget)

### 1.1. Макет (Layout) и Карточки (Panels)
**Проблема:** Нет визуального разделения виджетов. Графики и пончики сливаются.
**Решение:**
- [ ] **Сетка (Grid System):** Использовать `QGridLayout` для жесткого контроля пропорций.
- [ ] **Стиль Grafana-карточек:** Обернуть каждый график (Area Chart, каждый Donut) во фрейм (QFrame) с фоном `#1e1e2e` (чуть светлее основного фона `#11111b`), тонкой границей (`1px solid #313244`) и скругленными углами (`border-radius: 8px`).
- [ ] **Внутренние отступы:** Увеличить padding внутри карточек до 15-20px, чтобы контент не прилипал к краям.

### 1.2. Главный график трафика (Area Chart)
**Проблема:** Тонкие пересекающиеся линии (Line Series) тяжело читать при всплесках.
**Решение:**
- [ ] **Stacked Area Chart:** Преобразовать `QLineSeries` для TCP, UDP и ICMP в `QAreaSeries`.
- [ ] **Monium-стиль (Полупрозрачность):** Задать кистям (`QBrush`) альфа-канал (например, 40-50% прозрачности) для заполнения областей под линиями, создавая красивый перекрывающийся купол трафика. Убрать толстые рамки осей.

### 1.3. Кольцевые диаграммы (Donut Charts)
**Проблема:** В старом UI (судя по коду) они имели жесткую высоту `180px` и выглядели мелко.
**Решение:**
- [ ] **Резиновый размер:** Убрать `setFixedHeight`. Позволить диаграммам адаптивно занимать пространство карточки.
- [ ] **Текст в центре:** Реализовать отрисовку главного значения (например, "45%") крупным шрифтом прямо в "дырке" пончика (как в Grafana).
- [ ] **Легенда:** Переместить легенду (In Use / Free) вправо или аккуратно выровнять снизу с цветными квадратиками.

### 1.4. Умные Тултипы (Smart Tooltips a-la Monium)
**Проблема:** Стандартный тултип QtCharts показывает только одну точку для одной линии.
**Решение:**
- [ ] Написать кастомный класс (или переопределить события мыши в `InteractiveChartView`), который при наведении на график будет рисовать сводную HTML-карточку.
- [ ] **Формат карточки:** Столбик с перечислением: `[Цветной квадрат] TCP: 1000 pps`, `[Квадрат] UDP: 50 pps`. Внизу разделитель и строка `Сумма: 1050 pps`.

## 2. Раздел "Deep Analytics" и "Security Incidents"

### 2.1. Сетка Здоровья (AlertGridWidget / SLO)
**Решение:**
- [ ] Сделать сетку (массив зеленых/красных квадратов) более плоской, с отступами между ячейками (gap=4px) в стиле Heatmap GitHub/Grafana. 
- [ ] Добавить всплывающие подсказки (tooltips) к квадратам ("Timestamp: 12:05:00 - Status: OK").

### 2.2. Таблицы (Top Talkers, Log, Incidents)
**Проблема:** В старом UI таблицы выглядят слишком базово (стандартные серые заголовки).
**Решение:**
- [ ] **QSS Стилизация:** Убрать стандартную сетку (`showGrid(false)`), настроить высоту строк (`verticalHeader()->setDefaultSectionSize(35)`), сделать темный фон заголовков.
- [ ] **Прогресс-бары в ячейках:** (Особенно для "Top Talkers") - рисовать фоновую заливку ячейки пропорционально проценту IP-адреса от общего трафика.

### 2.3. Интерактивный Timeline (Uptime Bar)
**Решение:**
- [ ] Сглаживание блоков на таймлайне инцидентов (зеленые/красные отрезки) и добавление тултипов при наведении ("14:30 - DDoS Attack").
- [ ] При клике на красный блок таблица инцидентов должна фильтроваться по этому времени.

## 3. Глобальные улучшения (Toolbar, Header)

### 3.1. Header и Toolbar
**Проблема:** На старых скринах (`DDoS Dashboard*.png`) видна толстая или грубая зеленая полоса сверху. 
**Решение:**
- [ ] Сделать Header тонким (3-4px) и изящным акцентом (`#a6e3a1`).
- [ ] Кнопки в Toolbar ("Открыть PCAP") сделать в плоском стиле (Flat design) со сменой фона при наведении (Hover).

### 3.2. Звуковые уведомления
**Решение:**
- [ ] Интегрировать воспроизведение системного звука (`QApplication::beep()`) при переходе индикатора в статус "АТАКА".
